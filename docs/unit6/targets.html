    1. Use function notation, evaluate functions for inputs in their domains, and interpret statements that use function notation in terms of a context. Note: If f is a function and x is an element of its domain, then f(x) denotes the output of f corresponding to the input x.

  9. Define sequences as functions, including recursive definitions, whose domain is a subset of the integers.

    • Example: A sequence with constant growth will be a linear function, while a sequence with proportional growth will be an exponential function.
